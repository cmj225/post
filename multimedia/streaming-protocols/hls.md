# HLS

### HTTP Live Streaming

Apple에서 개발한 HTTP 기반 스트리밍 프로토콜

* 미디어 컨테이너 포맷 : .mp2ts\(.ts\), .mp4\(.fmp4\)
* menifest : .m3u8
  * 하나 또는 여러 비디오 파일의 경로를 plain text로 작성한 파일
  * Windows-1252, CP933 encoding
* 비디오 코텍 : h264
* aac
